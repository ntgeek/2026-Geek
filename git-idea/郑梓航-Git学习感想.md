# Git学习感想

我们每个人一出生，就相当于执行了git init，创建了自己的生命仓库。每天的经历，学到的新技能，做出的选择，都是一次git add，将变动放入暂存区。
而那些重大的决定毕业，工作，结婚就是一次次的git commit，附带着我们当时写下的日志(commit message)，记录了那一刻的理由与心情。

想尝试不同的人生道路?那就git branch一个新分支，去探索新的可能。也许这个分支最终会成功
merge回主线，让人生更丰富；也许它只是一个短暂的实验，最终被放弃，但依然留下了宝贵的经验。

犯了错，想反悔?生活很少允许我们git reset --hard，强行抹去过去。更多时候，我们能做的是 git revert，用一个新的"提交"来修正之前的错误，承认它的存在，然后继续前行，这或许就是成熟的代价。我们也不断面临 merge conflicts：事业与家庭的冲突，理想与现实的拉扯，都需要我们手动去解决，才能让生活继续。

最终，当我们回首往事，翻看人生的git log，会看到一条密密麻麻，独一无二的提交记录。它或许不完美，甚至有些混乱，但这正是我们自己一步步走来的证明，是我们完整而真实的一生。

以上内容来源于某博主，只补充一点：可惜人生不能git stash。
